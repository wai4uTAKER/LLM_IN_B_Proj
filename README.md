# LLM_IN_B_Proj
Проект демонстрирует построение поискового ассистента для маркетплейса на основе LLM и векторного поиска. Реализован полный пайплайн: разбор пользовательского запроса, рефразинг под индекс, извлечение кандидатов из FAISS, LLM-rerank и генерация объяснения результатов. В качестве данных используется датасет Avito Demand Prediction.

Данные
Используется датасет соревнования Kaggle:

Avito Demand Prediction Challenge
https://www.kaggle.com/competitions/avito-demand-prediction/data

Из датасета используется файл:
train.csv 

Подготовка данных
Из train.csv формируется единый текст для каждого объявления, который используется для семантического поиска. В текст входят название, описание, регион, город, категория и характеристики товара.

Векторизация
Для всех search_text считаются эмбеддинги с помощью sentence-transformers или совместимой модели. Эмбеддинги нормализуются и приводятся к float32.

FAISS индекс
На основе эмбеддингов строится FAISS индекс (IndexFlatIP). Индекс используется для быстрого семантического поиска по косинусному сходству. Метаданные объявлений сохраняются отдельно.
